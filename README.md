# FEARLESS MACRO

Fearless macro created by fearless9453, forked by cyhber.

This macro is a fishing macro for Blox Fruits, it works similar to a auto-clicker, **this is NOT a EXPLOIT/SCRIPT**

## Features

- [x] Functional fishing macro
- [x] Discord WebHooks logging
- [x] AFK Mode after the macro finishes
- [x] Chests (CURRENTLY IN DEVELOPMENT)
- [ ] Fishing Rod skills (TO ADD)
- [ ] AUTO OPENING CHESTS (TO ADD)

## Installation

1. Download this page from the download button or use **git**

```sh
git clone https://github.com/cyberspacepirates/fearless-macro
```

2. Use the most stable version

```sh
git checkout v2.3.0
```

3. Make sure you have **AutoHotKey** installed,

compile `fearless-autofisher.ahk` to **build/fearless-autofisher.exe**

## Windows Defender

If you let that macro running for hours, it's possible Windows Defender going to stop the execution and delete the executable, since the macro reads the screen, controls mouse and keyboard.

Add a exclussion to `fearless-autofisher.exe`

### Instructions

```js
Settings >
Privacy & Security >
Windows Security >
Virus & threat protection >
Manage settings >
Add or remove exclussions
```
